# GFP 蛋白设计亮度数据分析

本项目用于分析 `GFP_data.xlsx` 中的 GFP 突变体亮度数据，以及 `beforetopseqs` 中的候选序列。分析结果已按中文命名输出到 `分析结果/` 目录。

## 目录结构

```text
protein_design_analysis/
├── GFP_data.xlsx                         # 原始 Excel 数据
├── AAseqs of 5 GFP proteins.txt          # GFP 参考序列
├── README.md
├── 分析脚本/
│   └── GFP综合分析.py                    # 可重复运行的综合分析脚本
└── 分析结果/
    ├── GFP综合分析结果.xlsx              # 多工作表分析结果
    ├── GFP综合分析报告.html              # HTML 汇总报告
    └── 图表/                             # PNG 图表
```

## 数据内容

`GFP_data.xlsx` 包含两个工作表：

- `brightness`：141,572 条亮度记录，字段包括 `aaMutations`、`GFP type`、`Brightness`。
- `beforetopseqs`：20 条候选序列，2024 和 2025 各 10 条。

涉及的 GFP 类型：

- `avGFP`
- `amacGFP`
- `cgreGFP`
- `ppluGFP`

## 分析内容

当前脚本覆盖以下分析维度：

- 数据质量检查：缺失值、重复值、突变格式、候选序列长度、非法氨基酸字符。
- 亮度分布分析：各 GFP 类型亮度范围、分位数、直方图和箱线图。
- WT 对比：计算每条记录相对同类型 WT 的亮度变化。
- 检测下限识别：标记疑似检测下限记录。
- 突变数量分析：分析突变数量与亮度之间的关系。
- 最大亮度分析：按 GFP 类型和突变数量分层统计每层最高亮度，并计算最大亮度与突变数量的相关性。
- 位点价值分析：按突变数量校正后评估高价值、低价值位点。
- 具体替换价值分析：评估具体突变替换的关联收益或风险。
- 全类型合并分析：在每种 GFP 内先计算价值分数，再跨 GFP 类型等权汇总，识别跨类型高价值、低价值和分歧位点。
- 替换类型分析：统计不同氨基酸替换类型的整体表现。
- 多点突变共现分析：统计高频共现突变组合。
- 候选序列分析：比较候选序列与 `avGFP` 参考序列及 2024 首条序列的差异，统计年度突变频率变化。

## 关键分析口径

- `brightness` 中的突变位点采用 `0-based` 编号。例如 `A109D` 对应参考序列的常规第 110 位。
- 输出表中同时保留 `数据位点_0based` 和 `常规位点_1based`。
- 位点和具体替换价值使用“同 GFP 类型、同突变数量平均亮度”校正后的残差，减少多点突变数量造成的混杂。
- 最大亮度与突变数量分析按每个 GFP 类型、每个突变数量层取最高亮度；相关性同时报告全部层和样本数不少于 10 的层，以降低低样本极端值的误读风险。
- `高价值位点`、`低价值位点`、`高价值替换`、`低价值替换` 等不带“全类型”的表，是每一种 GFP 类型内部单独计算的结果。
- `全类型位点价值` 等带“全类型”的表，是先在每种 GFP 内计算分数，再跨 GFP 类型等权汇总的结果。
- 全类型位点按各参考序列的常规编号合并；同一编号可用于跨类型趋势筛选，但不保证是严格结构同源位点。
- 全类型替换类型分类要求覆盖至少 2 种 GFP，且总出现次数不少于 20。
- 位点高低价值分类要求位点出现次数不少于 30。
- 具体替换高低价值分类要求替换出现次数不少于 10。
- `*238G` 这类终止位点替换被作为特殊突变保留。

## 当前主要发现

- `avGFP` 存在明显疑似检测下限堆积，共 7,359 条记录亮度 `<= 1.30104`。
- 其他三类 GFP 未观察到同样明显的检测下限堆积。
- 在样本数不少于 10 的突变数量层中，`avGFP`、`amacGFP` 和 `cgreGFP` 的最大亮度随突变数量增加整体下降；`ppluGFP` 的单调关系不强。
- 候选序列结果保留相对 `avGFP` 参考和相对 2024 年首条序列的差异。

## 注意事项

高价值和低价值突变位点来自统计关联分析，用于候选筛选和实验优先级排序，不应直接解释为已经证明的因果效应。
